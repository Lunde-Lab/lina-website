# Lina Taglines

This document lists approved Lina taglines and how to use them.

## Approved Taglines

| Language       | Tagline                                        |
| -------------- | ---------------------------------------------- |
| Norwegian (NO) | Én oversikt - på tvers av hjem                 |
| Swedish (SV)   | En gemensam översikt - över hem                |
| Danish (DA)    | Én samlet oversigt på tværs af hjem            |
| Finnish (FI)   | Yhteinen kokonaiskuva yli kotien               |
| English (EN)   | One shared place for your child - across homes |

## Usage Guidelines

- Use taglines in brand-facing contexts (onboarding, landing pages, app store text, campaign headers).
- Keep the tagline text exact. Do not rewrite without approval.
- Use one language per surface.
- Use one tagline at a time.
- Pair taglines with calm, factual supporting copy.
- Do not add hype punctuation or emoji.

## Do/Don't for Taglines

| Do                                     | Don't                                     |
| -------------------------------------- | ----------------------------------------- |
| Use the approved wording exactly       | Rewrite the phrase into marketing slogans |
| Match tagline language to UI language  | Mix languages in one line                 |
| Keep visual treatment calm and clean   | Use playful or loud styling               |
| Place as a support line, not a command | Use as CTA text or button labels          |

## Recommended Placement

- Onboarding intro headline/subheadline
- App Store and website hero support line
- High-level product overview sections

Avoid using taglines in:

- Error messages
- Confirmation dialogs
- Destructive actions
- Dense operational UI states
