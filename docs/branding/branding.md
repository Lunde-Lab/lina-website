# Lina Branding

This document is the source of truth for Lina brand direction across product, content, and design.

## Brand Essence

### Core Idea

Lina is one safe line between two parents who share care.  
One place. No noise. Just collaboration.

### Brand Purpose

Create a calm, predictable, and secure communication surface for co-parenting, without distractions.

### Brand Promise

Everything about the child, safely gathered in one place.

### Brand Metaphor

A line that connects two people in a calm, stable, and structured way.

## Brand Pillars

### 1. Clarity

- Clean lines
- No clutter
- Minimalism without coldness
- Communication that is easy to read and understand

### 2. Safety

- Private data handling (RLS, private buckets, signed URLs)
- Calm and warm tone
- Non-pushy UI
- Respectful information handling

### 3. Collaboration

- Every feature supports two parents sharing care, regardless of conflict level
- Pair-based structure
- Tools that build predictability and shared understanding

## Tone of Voice

Lina sounds calm, safe, adult, neutral, and friendly.

### Communication Principles

- Keep messages short and clear
- Avoid hype words
- Avoid unnecessary exclamation marks
- Use warm language without becoming casual chat
- Give reassurance, not pressure

### Tone Do/Don't

| Do                                                 | Don't                             |
| -------------------------------------------------- | --------------------------------- |
| "Welcome. Here you get one safe line between you." | "Welcome!! This will be magical." |
| "Here is how to do it."                            | "You must do this now!"           |
| "We could not connect right now. Try again."       | "Oops! Something went wrong!"     |

## Visual Identity

### Color Direction

Color should support:

- Safety: calm teal and gray direction, no harsh contrast
- Adult quality: mature and professional feel
- Clarity: clean hierarchy without visual noise
- Calmness: muted tones that keep focus on collaboration

### Typography

- Font: Inter
- Weights: Regular, Medium, Semibold
- Principle: readability first, spacing over decoration, no "shouty" headings

### Icons

- Lucide icon style
- 20-24 px
- Simple line icons
- No filled icon style
- Route icon usage through `LinaIcon`

## UI Style

### Visual Principles

- Air and calm
- Horizontal lines as a supporting motif
- Balanced spacing
- Subtle shadows
- Soft radii (8 / 12 / 16)

### Component Direction

- `LinaButton`: clear, simple, safe
- `LinaText`: clean Inter typography
- `LinaSegmentedControl`: line metaphor in practice
- `Composer`: calm, neutral, adult

### UI Style Do/Don't

| Do                                                | Don't                                   |
| ------------------------------------------------- | --------------------------------------- |
| Use whitespace to reduce cognitive load           | Fill screens with decorative noise      |
| Use subtle state changes and low visual intensity | Use flashy animations and color flashes |
| Keep layouts predictable and structured           | Use playful, attention-seeking patterns |
| Design for collaboration workflows                | Design like a general social chat app   |

## Product Experience

### Threads

Core to Lina. Communication follows one clear line between the two caregivers.

### Album

Shared photo flow, safely stored and calmly presented.

### Tools

Tools support cooperation, not social engagement:

- Equipment list: practical things to remember (for example daycare items)
- About the child: birth date, allergies, medication, and custom notes
- Contact lists: pediatrician, teacher, and other key contacts
- Audit log: key events and who changed what, when

### Onboarding

- Focus on safety
- Show the line between two points
- Explain what Lina is, and what Lina is not

## Microinteractions

- Soft motion
- No bounce
- No color flashing
- Clear and simple state changes
- Use ease-in-out and short movement distances

## Content Guidelines

- Write short, clear, and friendly text
- Be informative without sounding commanding
- Be safe and steady without sounding passive

Preferred wording shifts:

- "Upload photo" -> "Add photo"
- "Delete permanently" -> "Delete photo" (explain consequences in helper text)
- "You must..." -> "You can..." or "Do this:"

## Brand Don'ts

- No childish color expression
- No pushy marketing language
- No gamification patterns
- No distraction-heavy UI
- Do not position Lina as a general-purpose chat app

## Brand Voice Quick Guide

Lina is:

- Calm
- Safe
- Adult
- Friendly
- Clear

Lina is not:

- Noisy
- Hype-driven
- Playful
- Ironic
- Loud
