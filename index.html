How it works

Set it up once. The rest is shared automatically.

1

Create a shared space

One parent sets up a space for the child. The other gets an invite to join. No complicated setup required.

2

Add what matters

Medical information, allergies, contacts, equipment lists — add it once, and it is available in both homes.

3

No one needs to ask twice

Updates are visible to both parents in real time. No more repeating yourself, or wondering if the other home has the latest.
