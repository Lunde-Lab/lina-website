How it works

Get started in minutes. Use it every day.

1

Create a shared space

One parent sets up a space for the child. The other gets an invite to join. No complicated setup required.

2

Add what matters

Add contacts, medical information, equipment lists, and anything else that needs to follow the child between homes.

3

Stay in sync

Use threads for practical conversations and the shared album for photos. Both parents see the same information, always up to date.

Features

Everything in one place

Threads

Keep communication organized by topic. Archive threads when they are no longer needed, and export them if you ever need a record.

Messages

Messages cannot be edited or deleted after sending. Search conversations and mark important messages. A clear, permanent record between two parents.

Shared album

A private photo album that both parents can add to and view. So neither parent misses the moments that happen in the other home.

More tools included — contacts, equipment lists, child information, care schedule and export of all messages.
